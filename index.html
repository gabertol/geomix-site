Simultaneous inversion of detrital zircon age distributions, heavy minerals, petrography and grain size — within a single coupled NNLS framework operating in simplex space.

Launch Web App Read Vignettes

Core model

X = A · B + ε
X — samples × features
A ∈ ΔK−1 — mixing proportions
B — endmember signatures
min ‖X − AB‖F²
s.t. A ≥ 0, ∑Ai = 1, B ≥ 0
Mixing is linear in simplex space. CLR transformation breaks this linearity — simplex+NNLS achieves ~17% lower MAE vs CLR+NNLS.

Complete reproducible analyses — from synthetic validation to real Cretaceous data from the Paraná Basin.

01

Method Validation — Synthetic Data (DUMMY4)

End-to-end benchmark on a synthetic dataset with known ground truth. Six methods compared: Baseline NNLS-simplex, Entropy+inv_f, DZ-only, BP-only, CLR+NNLS, CLR+OLS. Demonstrates the ~17% MAE advantage of simplex-space inversion over CLR approaches.

02

Non-Uniform Mixtures & A_local Diagnostics (DUMMY6)

Explores hydraulic sorting and grain-size-dependent fractionation using DUMMY6. Introduces the A_local diagnostic: block-specific mixing proportions are compared to the global A matrix to identify proxies carrying unique or inconsistent signal.

Supported data types

Any combination of proxy blocks

Distributional and compositional blocks are jointly inverted in a single model — each with its own B matrix, all sharing a common A.

⬡

Detrital Zircon U-Pb

Age distributions processed as kernel density estimates. Multi-variable (age + geochemistry) supported.

build_kde_block() · distributional
◈

Heavy Minerals

Point-counting proportions. Simplex-normalised and jointly inverted alongside other blocks.

build_compositional_block() · compositional
▦

Petrography (QFL)

Modal framework composition. Quartz-Feldspar-Lithic or any bulk petrology dataset.

build_compositional_block() · compositional
≈

Grain Size

Continuous distributions as KDE or binned fraction data as compositional vectors.

Installation

Install directly from GitHub. Requires R ≥ 3.5.0.

R
# Install from GitHub install.packages("devtools") devtools::install_github("gabertol/geomix") # Load library(geomix) # Minimal example fit <- unmix(data_list, K = 3) print(fit)
R · bootstrap
boot <- bootstrap_unmix( data_list = data_list, K = 3, n_boot = 200, conf_level = 0.95 ) plot(boot)
