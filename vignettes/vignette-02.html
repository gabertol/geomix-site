Non-Uniform Mixtures and Grain-Size Dependent Provenance

Overview

Standard provenance unmixing assumes that each sample is a uniform mixture of its sources — that is, the mixing proportions (A) are the same regardless of which analytical technique or grain-size fraction is examined. In practice, this assumption is often violated: different source areas supply sediment predominantly in different grain-size ranges, so the apparent contribution of each source shifts systematically across fractions.

This vignette demonstrates how geomix handles this scenario through the concept of local mixing proportions (A_local). Using the dummy6 synthetic dataset — where ground truth is fully known — we show that:

  1. A joint unmixing fit (A_global) recovers a single, fraction-averaged proportion per sample, missing the grain-size-dependent signal.
  2. A_local re-estimates proportions separately for each fraction by fixing the known end-member signatures (B) and re-solving for A per fraction, recovering the true mixing curves.
  3. The divergence between A_local and A_global is itself a diagnostic: large divergence signals non-uniform mixing; small divergence confirms the global assumption holds.

The DUMMY6 Dataset

dummy6 is a fully synthetic benchmark with known ground truth, designed to test the detection and reconstruction of grain-size-dependent mixing. It contains:

  • 15 samples (S01–S15), each a mixture of 3 end-members with varying proportions
  • 6 grain-size fractions: Very Coarse Sand (VCS), Coarse Sand (CS), Medium Sand (MS), Fine Sand (FS), Silt, and Clay
  • 6 minerals per fraction: Qtz, Fsp, Mica, Carb, Clay, Lith
  • End-member grain-size preferences: EM1 concentrates in coarse fractions, EM3 in fine fractions, EM2 intermediate — so the apparent mixing proportions shift predictably across fractions

Joint Unmixing: A_global

unmix() fits a single global A matrix — one set of proportions per sample, shared across all fractions. This is the standard unmixing result.

set.seed(123)

fit <- unmix(
  data_list  = data_list,
  data_types = data_types,
  K          = 3,
  weighting  = "inv_f",
  adaptive   = "none",
  verbose    = FALSE
)

# Align A_global to true EM labels via Hungarian algorithm
B_true <- dummy6$EM_mineralogy |>
  select(-EM) |>
  as.matrix()
rownames(B_true) <- dummy6$EM_mineralogy$EM
B_true   <- B_true[, minerals]
em_names <- rownames(B_true)

B_ref <- fit$B_list[[1]][, minerals]
cost  <- outer(
  seq_len(nrow(B_ref)), seq_len(nrow(B_true)),
  Vectorize(\(i, j) mean(abs(B_ref[i, ] - B_true[j, ])))
)
perm             <- as.integer(solve_LSAP(cost))
A_global_aligned <- fit$A[, perm]
colnames(A_global_aligned) <- em_names

Computing A_local

The key insight: fix the known end-member signatures (B) and re-solve for A independently within each fraction. This allows mixing proportions to vary across fractions while keeping the end-member identities fully consistent.

In DUMMY6, we use the ground-truth B directly to demonstrate that A_local perfectly recovers the true mixing curves when end-members are known. In real datasets, estimated B from the global fit would be used instead.

# Helper: solve A given fixed B, row-by-row with NNLS + simplex normalisation
solve_A_fixed_B <- function(X, B) {
  X  <- as.matrix(X)
  B  <- as.matrix(B)
  N  <- nrow(X)
  K  <- nrow(B)
  Bt <- t(B)
  A_est <- matrix(NA_real_, nrow = N, ncol = K,
                  dimnames = list(rownames(X), rownames(B)))
  for (i in seq_len(N)) {
    a_i <- coef(nnls(Bt, as.numeric(X[i, ])))
    s   <- sum(a_i)
    A_est[i, ] <- if (s > 0) a_i / s else rep(1 / K, K)
  }
  A_est
}

# Compute A_local for every fraction using true B
A_local_by_fraction <- fractions |>
  set_names() |>
  map(\(frac) {
    X <- blocks[[frac]]$data_mat[common_samples, ]
    solve_A_fixed_B(X, B_true)
  })

cat("A_local computed for:", names(A_local_by_fraction), "\n")

Validation: MAE by Fraction

A_true_by_frac <- fractions |>
  set_names() |>
  map(\(frac) {
    dummy6$A_per_fraction |>
      filter(fraction == frac) |>
      select(sample, all_of(em_names)) |>
      column_to_rownames("sample") |>
      as.matrix()
  })

mae_summary <- fractions |>
  map_df(\(frac) {
    A_loc  <- A_local_by_fraction[[frac]][common_samples, ]
    A_glob <- A_global_aligned[common_samples, ]
    A_true <- A_true_by_frac[[frac]][common_samples, ]
    tibble(
      fraction   = frac,
      MAE_local  = mean(abs(A_loc  - A_true)),
      MAE_global = mean(abs(A_glob - A_true))
    )
  }) |>
  mutate(fraction = factor(fraction, levels = fractions))

mae_summary |>
  pivot_longer(starts_with("MAE"), names_to = "strategy", values_to = "MAE") |>

Practical Guidance

When to use A_local:

  • When data blocks sample different grain-size fractions (e.g., detrital zircon in fine sand, bulk petrography in medium-coarse sand)
  • When you suspect sources differ in their grain-size distributions
  • As a diagnostic on any multiproxy fit: if A_local diverges strongly from A_global in a specific block, that block carries fraction-specific information beyond the global signal

Interpretation of divergence:

A_local vs A_global Interpretation
Small divergence (all blocks) Uniform mixing; global assumption holds
Large divergence (specific block) That proxy/fraction sees a different mixture
Systematic trend across fractions Grain-size-dependent provenance (this vignette)
